# Phoenix Capstone Project

A responsive React web application that explores our solar system through real-time planetary data. Built as a collaborative capstone project for the TS Academy Frontend Development Program, this project demonstrates React component architecture, Fetch API integration, responsive web design, form validation, Git workflows, and team collaboration.

## Live Demo

🔗 Deployment Link: **To be added after deployment**

## Project Summary

This application recreates the provided Phoenix Cohort Figma design as a fully functional React application.

### Features

- Fetches and displays real planetary data from an external API
- Responsive design for desktop, tablet, and mobile devices
- Dynamic planet visualization cards
- Comparative planetary data table
- Video section with autoplay, muted, and loop functionality
- Contact form with validation and submission handling
- Smooth scrolling CTA buttons
- Component-based React architecture
- CSS Modules for styling
- Collaborative Git workflow using feature branches and Pull Requests

## Technologies Used

- React
- JavaScript (ES6+)
- CSS Modules
- Fetch API
- Git & GitHub
- Vercel / Netlify (Deployment)

## Running the Project Locally

### 1. Clone the Repository

```bash
git clone https://github.com/JonesEgelle/group-28-frontend-capstone-project.git
````

### 2. Navigate Into the Project

```bash
cd group-28-frontend-capstone-project
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start Development Server

```bash
npm run dev
```

### 5. Open in Browser

```text
http://localhost:5173
```
